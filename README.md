What's "Pick!"?
-----------------
"小鸡接苹果"是利用Adobe AIR开发的Android游戏。
游戏的主要内容是控制小鸡接住从天而降的苹果。
资料库中有一个apk文件（版本并非最新），可以下载试玩。
"Pick!" is an android game, based on Adobe AIR platform.
You can try playing it by downloading from this repo.
